Python Mini Projects Collection

This repository contains a collection of practical Python mini projects developed during internship training. Each project demonstrates core Python concepts, file handling, basic security implementation, and simple web application logic.

The projects included in this collection are:

URL Shortener – Generates shortened links and stores mappings using JSON-based storage.

File Organizer – Automatically categorizes and organizes files based on their type.

Password Manager – Securely stores user credentials with basic encryption techniques.

Quiz Game – Interactive quiz system that reads questions from a file and tracks user scores.

These projects are built using Python and Flask (for web-based versions) and showcase fundamental programming concepts such as:

File handling

JSON data storage

Basic encryption and hashing

User input handling

Simple web development using Flask

This collection serves as a demonstration of applied Python programming and problem-solving skills.
